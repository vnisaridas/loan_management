## Usage

### Installation

Install the dependencies

```sh
$ composer install
```

Generate project key 

```sh
$ php artisan key:generate
```

Change env file
```
$ .env.example to .env
```
migrate database

```sh
$ php artisan migrate
```

seed database

```sh
$ php artisan db:seed
```

start developement server

```sh
$ php artisan serve
```
